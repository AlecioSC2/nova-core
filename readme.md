# Nova Core
Small templating framework that decouples the presentation layer with the business logic. Based on XumaK's Danta Framework specification.

## Run example
```bash
node examples/example.js
```

## Run Tests
```bash
npm test
```

## Generate API docs
```bash
npm run jsdoc2md
```
